PHP Form Project

This is a simple PHP form project where users can enter data and submit it. The form data is processed using PHP.

Features

Simple HTML form

PHP script to handle form submission

Displays submitted data

Easy to understand for beginners

How to Run

Install XAMPP or WAMP.

Place the project folder inside the htdocs (XAMPP) or www (WAMP) folder.

Start Apache server.

Open browser and run:

http://localhost/phpform/


Fill the form and submit.

File Structure
project/
│── index.php        # HTML form
│── submit.php       # Handles form data

Requirements

PHP 7+

Web browser

Local server (XAMPP/WAMP)
